# Reborn-Form-Hell
A 2D pixel horror puzzle game. You explore a dark place and solve simple puzzles. Find clues, avoid danger, and discover a scary story.

## 👻 Overview
Reborn: Form Hell is a **2D pixel horror puzzle game** where players explore a dark, mysterious environment filled with danger and hidden secrets.  
You must **solve puzzles, avoid enemies, and uncover a disturbing story** hidden deep within the game world.
